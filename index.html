Estrutura do HTML

(Voltar)

A estrutura básica do HTML é:

<!DOCTYPE html>
<html>
<head>
<body>

Esse é o básico da estrura

Tags de texto

(Voltar)

As principais tags de texto são:

<h1> - <h2> - <h3> - são usados para títulos e quanto maior o número menor a letra
<p> - que é parágrafo.

Listas ordenadas e não ordenadas

(Voltar)

Existem dois tipos de fazer listas com o HTML: ordenada e não ordenada

<ol> - Listas ordenadas, ela enumera a lista(1,2,3...).
<ul> - Lista não ordenada, ela faz com que a lista fique com marcadores, que podem ser alterados depois no CSS.

Mais tags

(Voltar)

Existem muitas outras tags, mas a tag que mais gostei de conhecer foi a tag <input>

Essa tag faz as caixas clicáveis para escrever, como as de formulário..
